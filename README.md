# DbMiru

DbMiru is a lightweight, developer-focused desktop database client built with Rust and gpui.

## Goals

- Fast startup
- Simple SQL execution
- Clear results and errors
- Designed for future multi-database support (starting with PostgreSQL)

## Status

Early development (M1)

See `AGENTS.md` and `docs/` for requirements, milestones, UI, and architecture.
